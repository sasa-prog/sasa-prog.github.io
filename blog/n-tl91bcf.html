Githubへのpushの仕方

※身内へ向けて書いてます。正確ではないです

1.レポジトリを作る
2.コマンドプロンプト or powershell or ターミナルで次のコマンドを実行

$ git clone https://github.com/ユーザー名/レポジトリ名
$ cd レポジトリ名

こうすることで作業ディレクトリへ移動
3.開発
4.一区切りついたらコミットする
コミットとは追加したファイルをgitに登録することだと思ってください。

$ git commit -m "コミットの内容を表すメッセージ"

5.リモートレポジトリへpush
　pushはリモートのレポジトリに変更を反映することです

$ git push origin main

こうすることでリモートに変更が反映されます。
